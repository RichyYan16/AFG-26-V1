## Example Usage Flow

```typescript
// 1. Get diagnosis
const diagnosis = await diagnoseWithHybridModel({
  understandsQuestion: "no",
  canSubmitBadInFiveMinutes: "no",
  strongestEmotion: "overwhelmed",
  taskScope: "large_clear",
  gradeWorry: "high"
});

// 2. Display diagnosis (SPEC EXACT FORMAT)
// - rankedTypes (HIGH to LOW) ✅
// - summary (VERY BRIEF) ✅
// - "Do you want to change this?" button
// - "What's the plan?" button

// 3. (Optional) Show internal follow-up questions
diagnosis.internalFollowUpQuestions.forEach(q => console.log(q));

// 4. Get intervention plans when "What's the plan?" clicked
const plans = await buildMultipleInterventionPlans(diagnosis.primaryType);

// 5. Display plans (MULTIPLE IDEAS)
plans.forEach((plan, i) => {
  console.log(`Plan ${i+1}: ${plan.headline}`);
  console.log(plan.whyItWorks);
  plan.steps.forEach(step => console.log(`${step.timeMinutes}min: ${step.action}`));
});
```
