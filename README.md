Unstuck
<br>
An app to help students overcome academic paralysis and approach school with confidence
<br>
To access the app, clone the repo in the terminal with the command "git clone https://github.com/Abhi52897340598327/AFG-26-V1.git", install node modules with the command "npm install", and then run "npm run dev", which loads a server for the app to run on. Type "localhost:3000" in the search bar of your preferred browser, and then you will be able to use the app.
